# GitHub social preview checklist

After this branch is merged:

1. Open the repository on GitHub and choose **Settings → General**.
2. Find **Social preview**, choose **Edit**, and upload `braid-social-preview.png`.
3. Confirm the 1280 × 640 image is readable in the preview, then save.
4. Re-open the repository in a private browser window and verify the preview card.

The committed file does not change the GitHub repository setting by itself. Do not claim the setting is
updated until the repository owner completes and verifies these steps.
